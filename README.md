# ✨ Magic English: Тренажер английских слов 🎓

![Banner](https://img.freepik.com/free-vector/learning-concept-illustration_114360-1107.jpg)

**Magic English** — это современное веб-приложение на базе Python, созданное для эффективного запоминания новых английских слов с использованием метода случайных карточек и надежного хранения данных.

---

## 🚀 Основные возможности

* **📥 Мгновенное добавление:** Удобная форма для записи новых слов с выбором категории.
* **🧠 Умная тренировка:** Режим карточек, который помогает повторять слова в случайном порядке.
* **🗂️ Цифровой словарь:** Полный список всех сохраненных слов в виде интерактивной таблицы.
* **🎨 Красивый интерфейс:** Современный градиентный дизайн с адаптивной версткой.

---

## 🛠 Технологический стек

| Технология | Использование |
| :--- | :--- |
| **Python 3.10+** | Основной язык разработки |
| **Streamlit** | Создание интерактивного интерфейса и дизайна |
| **SQLite3** | Локальная база данных для надежного хранения слов |
| **Pandas** | Обработка и визуализация данных словаря |

---

## 🤖 AI-Usage (Использование ИИ)

В ходе разработки использовался ИИ (Gemini) для:
1.  Проектирования архитектуры базы данных SQLite.
2.  Написания кастомных CSS-стилей для создания уникального дизайна (градиенты, стеклянные карточки).
3.  Оптимизации логики переключения карточек через `st.session_state`.
