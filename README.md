# English Word Tracker 🎓

Приложение для изучения английских слов.

## Функционал
- **Добавление слов:** Сохранение слов и перевода в базу SQLite.
- **Тренировка:** Генерация случайных флэш-карт.
- **Словарь:** Просмотр списка всех слов.

## AI-usage
ИИ (Gemini) помог с логикой SQLite и структурой Streamlit.Я настроила интерфейс и валидацию данных.

## Как запустить
1. `pip install streamlit pandas`
2. `streamlit run app.py`
